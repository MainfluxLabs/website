MAINFLUX MFX-1 INDUSTRIAL EDGE GATEWAY

The MFX-1 IoT Edge Computing Gateway combines Mainflux IoT platform with optimized
SolidSense N6 industrial hardware, delivering a robust solution for local IoT networks
that meets stringent industrial requirements for performance, security, and reliability.

MFX-1 EDGE GATEWAY FEATURES

SOFTWARE PLATFORM

Full Mainflux IoT Platform capabilities



  • Enterprise-grade security with Mutual TLS Authentication (mTLS) using X.509 certificates


  • Protocol and device agnostic architecture with multi-protocol support


  • Highly scalable from constrained devices to cloud deployments


  • High performance through Golang implementation - lightweight, statically compiled with minimal memory footprint


  • Comprehensive remote device management for secure operation of distributed IoT networks


  • Unified Cloud/Edge architecture enabling seamless management and deployment of complex IoT solutions
HARDWARE PLATFORM

The MFX-1 gateway utilizes Solid Run HummingBoard-Edge hardware with industrial-grade features:

  • Optimized memory footprint and binary size

  • Low latency, high performance architecture

  • Modular expansion capabilities

  • NXP i.MX6 Arm Cortex A9 System-on-Module

  • Optional AI accelerator and TPM 2.0 security

  • Industrial temperature range (-25° to 65°C)

  • Multiple connectivity options: Ethernet, WiFi, Bluetooth, GPS, LTE

  • Dual LTE modem support (Dual-SIM or Dual Modem)

UNIFIED CLOUD/EDGE ARCHITECTURE

The MFX-1 gateway shares the same codebase with Mainflux cloud platform, creating
a unified IoT implementation across the edge-fog-cloud continuum for simplified deployment
and management compared to maintaining separate platform instances.

The Mainflux IoT Platform serves as the centralized management system for edge gateways,
providing remote control, advanced analytics, and long-term data storage capabilities.

When deployed on gateways, Mainflux handles device connectivity across multiple protocols,
implements security through authentication and encryption, and enables local data processing
with lightweight database storage when needed.
